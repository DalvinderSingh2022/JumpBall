# JumpBall Game

Welcome to JumpBall, a simple and addictive 2D platformer game where you control a bouncing ball trying to reach new heights! 
Test your reflexes and timing skills as you navigate through various obstacles and platforms.


## Gameplay

JumpBall is all about jumping and bouncing your way up while avoiding obstacles.
The goal is to reach the highest point possible and set a new high score. 
Be cautious â€“ one wrong move can send you plummeting down!

## Features

- **Simple and Addictive**: JumpBall offers easy-to-understand gameplay that becomes progressively challenging, keeping you engaged.
- **Dynamic Obstacles**: Dodge moving platforms, spikes, and other obstacles that stand in your way.
- **High Score**: Compete against yourself and friends to achieve the highest possible score.
- **Vibrant Graphics**: Enjoy colorful visuals and a lively game environment.
- **Levels and Skin**: Enjoy different levels(maps) along with some awesome skins for ball.
